Справка по приложению

1. Главное окно «Транзакции»

Описание

  • Отображает все транзакции (продажи и заказы).
  • Фильтры: показать «Все», только «Продажи» или только «Заказы».
  • Двойной клик по строке — открыть детали транзакции.

В деталях транзакции

  • Можно выполнить возврат товаров, выбрав нужные позиции.
  • Возврат недоступен, если заказ ещё не «Завершён».
  • После выбора возврата — обновление в кассе и история возвратов.

2. Окно «Товары»

  • Список всех товаров с колонками: Название, Бренд, Количество.
  • В колонке «Действия» — кнопки «Редактировать» и «Удалить».
  • Добавление категории и товара через кнопки сверху.
  • Сортировка: «Все», «Часто продаваемые», «Нечасто».
  • Поиск по любым полям таблицы.

3. Окно «Продажи»

  • Показывает товары с их доступным количеством.
  • Фильтры: «Все», «Часто продаваемые» и поиск.
  • В колонке «Количество» — кнопки «+»/«–» для заказа.
  • Скидка вводится либо в BYN, либо в процентах.
  • При необходимости — ввод сдачи («Сдача с»).
  • Кнопка «Подтвердить заказ» вызывает окно подтверждения:
    • Показывает состав заказа, акции, скидки, итог и сдачу.
    • Выбор «Оплатить» или «Отменить».
  • Кнопки «Шаблон» (скачать Excel-шаблон) и «Excel-заказ» (загрузить заполненный файл).
  • При загрузке Excel-файла открывается окно оформления заказа.

4. Окно «Заказы»

  • Список текущих заказов с их статусами.
  • Статусы: «Оформлен», «Собран», «В пути», «Доставлен», «Завершён», «Отменён».
  • Статус «Завершён» можно установить только после подтверждения доставки.
  • При изменении статуса клиенту отправляется уведомление на почту.

5. Окно «Касса»

  • Ввод суммы и выбор операции: «Пополнение» или «Снятие».
  • Кнопка «Выполнить» — проводит операцию.
  • Кнопки «История операций» и «История возвратов» для соответствующих отчётов.
  • В «Истории операций» доступны экспорт в Word и Excel.

6. Окно «Скидки / Акции»

  • Список всех текущих акций и уровней лояльности.
  • Кнопки: «Добавить акцию», «Редактировать» и «Удалить» (предварительно выбрать запись).
  • Управление уровнями лояльности аналогично (отдельная вкладка).

7. Окно «Отчёты»

  • Поля: «Начальная дата», «Конечная дата», «Тип отчёта».
  • Кнопка «Сформировать отчёт» генерирует данные на указанный период.
  • Можно экспортировать результат в Excel, Word или PDF.

8. Окно «Персонал»

  • Поле для ввода уникального кода менеджера.
  • Кнопки: «Создать код», «Просмотреть мой код».
  • Список менеджеров, зарегистрированных под вашим кодом.
  • Менеджер при входе использует свой уникальный код.
